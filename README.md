# 📷 SnapSense AI

## Intelligent Screenshot Analyzer using OCR

SnapSense AI is a Python-based web application developed using Streamlit that analyzes screenshots intelligently. It extracts text using OCR, identifies the screenshot type, generates a summary, extracts important information, and allows users to download an analysis report.

---

## Features

- 📄 OCR Text Extraction
- 🧠 Screenshot Type Detection
- 🤖 AI-style Summary
- 🔍 Information Extraction
- 📊 Dashboard Metrics
- 📥 Download Analysis Report

---

## Technologies Used

- Python
- Streamlit
- EasyOCR
- NumPy
- Pillow
- Regular Expressions (re)

---

## Project Workflow

1. Upload Screenshot
2. OCR extracts text
3. Detect screenshot type
4. Generate summary
5. Extract important information
6. Display dashboard
7. Download report
